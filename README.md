# Java Crash Course Prep (BOA Global Tech)

This repository is a lightweight Java learning workspace focused on core syntax, compilation, and execution.

## Quick Start

1. Create a source file:

   ```java
   // src/Main.java
   public class Main {
       public static void main(String[] args) {
           System.out.println("Hello, Java");
       }
   }
   ```

2. Compile:

   ```bash
   javac -d out src/Main.java
   ```

3. Run:

   ```bash
   java -cp out Main
   ```

See `md/java-basics.md` for a short command/reference guide.
